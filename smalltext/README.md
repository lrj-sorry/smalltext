# SmallText 网站

这是一个简单的网站，具有登录功能和背景音乐播放功能。

## 功能特性

- 美观的登录界面
- 用户验证功能
- 登录后自动播放背景音乐
- 音乐播放控制（播放/暂停）
- 响应式设计，适配各种屏幕尺寸

## 文件结构

```
smalltext/
├── index.html      # 主HTML文件
├── styles.css      # 样式表文件
├── script.js       # JavaScript功能文件
├── background-music.mp3.txt  # 音乐文件说明（需替换为实际MP3文件）
├── d3cbfa9181638bfb3fce57ad465f6f34.mp3  # 背景音乐文件
└── README.md       # 项目说明文件
```

## 如何使用

1. **准备背景音乐文件**：
   - 下载一个MP3格式的背景音乐文件
   - 将文件重命名为 `background-music.mp3`
   - 放在与index.html相同的目录下

2. **打开网站**：
   - 直接在浏览器中打开 `index.html` 文件

3. **登录系统**：
   - 使用以下测试账号登录：
     - 用户名: admin，密码: admin123
     - 或 用户名: user，密码: user123

4. **背景音乐控制**：
   - 登录后音乐会自动尝试播放（取决于浏览器策略）
   - 可以使用主页面上的按钮控制音乐播放/暂停

## 注意事项

- 由于浏览器策略限制，某些浏览器可能不会自动播放音频，需要用户交互才能开始播放
- 本项目使用的是前端简单验证，实际生产环境中应使用后端验证确保安全性
- 推荐使用现代浏览器（Chrome、Firefox、Edge等）以获得最佳体验
